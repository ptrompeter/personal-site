Scotland Yard

An implementation of Scotland Yard, the classic board game, programmed in Python on the Django framework. Playable by two players.

Mars Street View

Mars Street View accesses and displays photos from NASA's three Mars rovers. Because NASA limits the number and size of calls to its APIs, our app periodically downloads and saves links to individual photos, then organizes these photos to allow users to follow the course of rovers across mars.

Sortable by rover, date, or even camera.

Developed in Python using the Pyramid framework. A Javascript front-end accesses the photos via AJAX.

SpeakEasy

Speakeasy is a web-messaging app that allows registered users to send messages to each other; these messages are automatically translated via the google translation API. It also allows users to communicate via chatroom.

Javascript with a Node backend.

RacTrac

RacTrac is a simple app to that allows Seattle bicyclists to find nearby bike racks maintained by the city of Seattle using the Google Maps API. The user's position is then compared to available bike rack location data to find the nearest rack. Bike rack data courtesy City of Seattle.

Javascript with a simple node backend to preserve secrets.
